NeuroMap is a parent-powered review platform for autism-friendly and sensory-friendly public environments.

This repository contains a static frontend implementation (no backend). Seed place + review data is bundled from `src/data/neuromap_la_jolla_places.json`, and any reviews you add are saved locally in the current browser via `localStorage`.

See `README.md` for setup + deployment, and `DESIGN_NOTES.md` for how the logo informed the visual system.
